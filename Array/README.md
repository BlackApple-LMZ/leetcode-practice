### 31 给了一个vector，找出下一个组合数 ** top 好题
```
1,2,3 → 1,3,2 
3,2,1 → 1,2,3
1,1,5 → 1,5,1
```
思路：原地转换，常量的转换时间；原地快排：先找到需要排序的区间，然后对改区间进行排序，自己实现的这种方法，中间有很多判断需要考虑，4ms也还行；solution O(n)的时间复杂度：和我的思路差不多，只不过不需要对改进区间重新排序，因为交换之后是由大到小排序的，只需要将这块区间逆序就行！！！妙啊
