**873 求一段序列里面最长的斐波那契子序列 

思路：开始想法是对的，用二维dp，i，j代表以ij为结尾的子数列，使用hash也想到了，但是开始不敢实现，后来发现大佬们的想法也差不多，就用于实现，最后写出来了。这个DP[i][j]数组的含义是，以i和j为结尾两个数字的费布拉奇数列长度（i < j）。条件是 A[i] + A[j] = A[k]。

**413 求一段序列里面等差数列的数目 长度必须大于3 连续的 这个题很好了！！！
