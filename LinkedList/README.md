*2. 链表表示的两个数求和 

Input: (2 -> 4 -> 3) + (5 -> 6 -> 4) 
Output: 7 -> 0 -> 8
Explanation: 342 + 465 = 807

思路比较简单，就是模拟加法的过程 考虑进位的情况

**19. 移除链表倒数第n个节点 

Given linked list: 1->2->3->4->5, and n = 2  Output 1->2->3->5.

思路: 先求出length，然后移除第length-n+1个node;用快慢指针，开始快指针先前移n个位置，然后和慢指针一同移动;
